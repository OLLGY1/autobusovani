# Informační systém - Autobusování/Šalinování

<h2>POPIS</h2>

Aplikace pro lidi co pravidelně cestují autobusama/šalinama a neví, jestli a kdy se mají postavit pro uvolnění místa a kdy ne.

<h2>CÍL SYSTÉMU</h2>

Cíl aplikace by byl zjednodušit miskomunikaci mezi lidmi v šalinách/autobusech ohledně zasedání

<h2>CÍLOVÁ SKUPINA</h2>

Každý člověk, který pravidelně jede autobusem/šalinou

<h2>BĚŽNÝ UŽIVATEL</h2>

Bežny uživatel si po založení profilu může informace na profilu měnit, také se může dívat na list lidí s aplikací podle autobusu/šaliny a jak moc si hodlají odsednout či jak moc si chcou sednout.
Každej autobus/šalina by měla svůj individuální list, který by se sám měnil velice často.

<h2>ADMIN</h2>

Admin může přidávat nové autobusové/šalinové trasy (a k nim listy) nebo měnit cesty podle toho, co je kde rozkopané.

<h2>SPRAVOVANÁ DATA</h2>

List autobusu (Číslo autobusu, na jakou stranu vede, kolik lidí v něm mají aplikaci)
Uživatel (Věk, sedí/nesedí, chce si sednout/nechce si sednout/je jim to jedno)
